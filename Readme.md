# Api rest typescript + typeorm

## Prérequis


1. Nodejs


## Instructions


1. `npm i` pour installer les dépendances.
2. `npm start` pour démarrer l'application.

## Variables .env

```
    TYPEORM_CONNECTION=
    TYPEORM_HOST=
    TYPEORM_USERNAME=
    TYPEORM_PASSWORD=
    TYPEORM_DATABASE=
    TYPEORM_PORT=
    TYPEORM_SYNCHRONIZE=
    TYPEORM_LOGGING=
    TYPEORM_ENTITIES=
```
